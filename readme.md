#simple store REST API

Built with Flask, Flask-JWT, Flask-RESTful, and Flask-SQLAlchemy
Deployed on Heroku - https://store--rest.herokuapp.com/

End points:
'/item/<string:name>' - get, put, post, del
'/store/<string:name>' - get, put, post, del
'/items' - get
'/stores' - get
'/register' - post, auth
